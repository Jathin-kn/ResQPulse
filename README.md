Project Name  
 MyOSA LifeSaver – IoT-Based CPR Assistance Prototype

Short Description  
 MyOSA LifeSaver is a modular embedded systems prototype built to explore how sensing, motion control, and real-time feedback can work together in CPR assistance system engineering. The project is designed as a learning and experimentation platform, bringing together hardware, firmware, and mechanical design in a clear and responsible way.

Purpose  
 This prototype was created to help students and researchers understand the building blocks of CPR assistance systems. Instead of focusing on outcomes, it emphasizes transparency, predictability, and hands-on learning through controlled experimentation.

Key Highlights

* Modular and easy-to-understand system architecture

* Sensor-based monitoring of orientation, movement, and interaction

* Controlled motor-driven motion inspired by CPR compression cycles

* On-device OLED display for live system feedback

* Designed for academic learning and demonstrations

Non-Clinical Note  
 This project is non-clinical and intended only for learning, experimentation, and demonstration purposes.

Tech Overview

* Controller: ESP32 (MYOSA custom motherboard)

* Sensors: MPU6050, BMP180, APDS9960

* Display: SSD1306 OLED

* Actuation: DC motor with motor driver

* Firmware: Embedded C (Arduino framework)

Getting Started

1. Power the system

2. Allow sensors to initialize

3. Observe live system information on the OLED display

4. Run demonstrations using the predefined motor cycle

5. Stop the system manually after testing
